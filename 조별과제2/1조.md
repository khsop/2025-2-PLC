# 전기공압 제어 과제 (1조)

## 1. 과제 목표
- 버튼 **PB1**: 1회 ON/OFF → **A+, A-, B+, B-** 자동 순서
- 총 **3사이클** 반복 후 정지
- 버튼 **PB2**: 카운터 리셋
- ES1 비상정지: 즉시 정지 및 풀리셋
- 유량밸브: 속도 약 50%

---

## 2. 사용 기호와 역할
- **LS1~LS4** : 실린더 전·후단 감지 리미트 스위치
- **Y1, Y2** : 솔레노이드 밸브 (A, B 실린더 전후진 제어)
- **K1~K5** : 릴레이 단계 제어
- **C1** : 카운터 (3사이클 후 정지)
- **ES1** : 비상정지 스위치
- **유량밸브** : 속도 조절

## 5. 변위단계선도
- A 실린더: 1단계 전진, 2단계 후진
- B 실린더: 3단계 전진, 4단계 후진
- 4단계 = 1사이클 → 3회 반복 후 종료

## 6. 단계별 동작 설명

| 단계 | 동작 | 구동밸브 | 전이 센서 | 비고 |
|------|------|----------|-----------|------|
| 1    | A+   | Y1 전진 | LS2 | 시작 조건 C1 < 3 |
| 2    | A-   | Y1 후진 | LS1 | |
| 3    | B+   | Y2 전진 | LS4 | |
| 4    | B-   | Y2 후진 | LS3 | C1 = C1+1 |

---

## 7. 카운터와 운전 조건
- PB1: 연속운전 → C1 < 3 동안만 진행
- C1 = 3 → 동작 정지
- PB2: C1 초기화(0으로 리셋)

---

## 8. 속도 조정
- 스로틀 밸브를 절반 정도 열어 왕복 속도 50% 근처로 맞춤

---

## 9. 시운전 절차
1. 초기 상태: A, B 후진단(LS1, LS3 ON) 확인
2. PB1 ON → 자동 운전 시작
3. A+, A-, B+, B- 순서 확인
4. 총 3사이클 후 정지 확인
5. 이상 시 ES1로 긴급 정지

---

## 10. 트러블슈팅
- **단계 멈춤** → 해당 센서 신호(LS) 및 릴레이(K) 확인
- **카운터 불량** → LS3 및 C1 배선 확인
- **속도 문제** → 유량밸브 미세 조정
